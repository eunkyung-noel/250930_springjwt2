# 05. ES6+ 주요 문법

#es6 #ecmascript #이씨엠에이스크립트

ES6(ECMAScript 2015)는 JavaScript에 많은 중요한 개선 사항을 도입했습니다. 이 버전부터 매년 새로운 기능이 추가되면서 JavaScript는 더욱 강력하고 현대적인 언어로 발전하고 있습니다. 여기서는 개발에 필수로 여겨지는 ES6+의 주요 문법들을 소개합니다.

---

## 1. 템플릿 리터럴 (Template Literals)

#템플릿리터럴 #template-literals

백틱(`` ` ``)을 사용하여 문자열을 정의하는 방식입니다. 여러 줄의 문자열을 쉽게 작성할 수 있고, `${}`를 통해 문자열 안에 변수나 표현식을 간편하게 삽입할 수 있습니다.

```javascript
const name = "홍길동";
const age = 30;

// 이전 방식
const message1 =
  "안녕하세요, 제 이름은 " + name + "이고, 나이는 " + age + "살입니다.";

// 템플릿 리터럴 방식
const message2 = `안녕하세요, 제 이름은 ${name}이고, 나이는 ${age}살입니다.`;

console.log(message2);
// 출력: 안녕하세요, 제 이름은 홍길동이고, 나이는 30살입니다.

// 여러 줄 문자열
const multiLine = `이것은
여러 줄로 이루어진
문자열입니다.`;
console.log(multiLine);
```

## 2. 구조 분해 할당 (Destructuring Assignment)

#구조분해할당 #destructuring #디스트럭처링

배열이나 객체의 속성을 분해하여 그 값을 개별 변수에 쉽게 담을 수 있게 해주는 문법입니다.

### 가. 객체 구조 분해

```javascript
const person = {
  firstName: "Gildong",
  lastName: "Hong",
  age: 30,
};

// 객체의 속성을 같은 이름의 변수에 할당
const { firstName, age } = person;

console.log(firstName); // "Gildong"
console.log(age); // 30

// 변수 이름을 다르게 지정하고 싶을 경우
const { firstName: fName, lastName: lName } = person;
console.log(fName); // "Gildong"
```

### 나. 배열 구조 분해

```javascript
const colors = ["red", "green", "blue"];

// 배열의 요소를 순서대로 변수에 할당
const [firstColor, secondColor] = colors;

console.log(firstColor); // "red"
console.log(secondColor); // "green"

// 특정 요소 건너뛰기
const [, , thirdColor] = colors;
console.log(thirdColor); // "blue"
```

## 3. 스프레드/나머지 연산자 (Spread/Rest Operator)

#스프레드연산자 #spread-operator #나머지연산자 #rest-operator

점 세 개(`...`)를 사용하는 연산자로, 사용되는 위치에 따라 "스프레드(펼치기)" 또는 "나머지(모으기)"의 역할을 합니다.

```mermaid
graph TD
    subgraph "스프레드 (Spread): 펼치기"
        A[배열/객체] --> B["... 연산자"];
        B --> C[개별 요소들로 분리];
        C --> D[새로운 배열/객체 생성, 함수 인자 전달 등에 사용];
    end

    subgraph "나머지 (Rest): 모으기"
        E[여러 요소들] --> F["... 연산자"];
        F --> G[하나의 배열로 합치기];
        G --> H[함수 매개변수, 구조 분해 할당 등에서 사용];
    end
```

### 가. 스프레드 연산자 (Spread Operator)

배열이나 객체의 모든 요소를 개별적으로 "펼쳐서" 다른 배열이나 객체에 넣거나 함수의 인자로 전달할 때 사용합니다.

**배열 복사 및 병합:**

```javascript
const arr1 = [1, 2, 3];
const arr2 = [4, 5, 6];

// 배열 복사 (얕은 복사)
const copiedArr = [...arr1];
console.log(copiedArr); // [1, 2, 3]

// 배열 병합
const mergedArr = [...arr1, ...arr2];
console.log(mergedArr); // [1, 2, 3, 4, 5, 6]
```

**객체 복사 및 병합:**

```javascript
const obj1 = { a: 1, b: 2 };
const obj2 = { c: 3, d: 4 };

// 객체 병합 (같은 키가 있으면 뒤의 값으로 덮어씀)
const mergedObj = { ...obj1, ...obj2, b: 99 };
console.log(mergedObj); // { a: 1, b: 99, c: 3, d: 4 }
```

### 나. 나머지 연산자 (Rest Operator)

주로 함수의 매개변수나 구조 분해 할당에서 여러 개의 요소를 하나의 배열로 "모아서" 받을 때 사용합니다.

**함수 매개변수:**

```javascript
// ...args는 전달된 모든 인자를 배열로 받음
function sum(...args) {
  return args.reduce((total, current) => total + current, 0);
}

console.log(sum(1, 2, 3)); // 6
console.log(sum(10, 20, 30, 40)); // 100
```

**구조 분해 할당:**

```javascript
const numbers = [10, 20, 30, 40, 50];

// 첫 번째와 두 번째 요소를 제외한 나머지를 'rest' 배열에 할당
const [first, second, ...rest] = numbers;

console.log(first); // 10
console.log(second); // 20
console.log(rest); // [30, 40, 50]
```
