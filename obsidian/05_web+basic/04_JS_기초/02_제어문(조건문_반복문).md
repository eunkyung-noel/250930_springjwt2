# 02. 제어문 (조건문, 반복문)

제어문(Control Flow Statement)은 코드의 실행 흐름을 특정 조건이나 반복에 따라 제어하는 구문입니다. JavaScript는 다른 프로그래밍 언어와 유사한 조건문과 반복문을 제공합니다.

#학습목표

- `if...else`와 `switch`를 사용하여 조건에 따라 다른 코드를 실행할 수 있습니다.
- `while`과 `for`를 사용하여 특정 코드를 반복 실행할 수 있습니다.
- `for...of`와 `for...in`의 차이점을 이해하고 각각 배열과 객체를 순회하는 데 사용할 수 있습니다.

---

## 1. 조건문 (Conditional Statements)

### `if`, `else if`, `else`

가장 기본적인 조건문입니다. 주어진 조건식(condition)이 `true`로 평가되면 코드 블록을 실행합니다.

```javascript
const score = 85;

if (score >= 90) {
  console.log("A 등급");
} else if (score >= 80) {
  console.log("B 등급"); // 이 블록이 실행됩니다.
} else {
  console.log("C 등급 이하");
}
```

- 조건식에는 Truthy/Falsy 값도 사용될 수 있습니다.
  ```javascript
  let username = prompt("이름을 입력하세요:"); // 사용자가 아무것도 입력하지 않으면 "" (falsy)
  if (username) {
    console.log(`안녕하세요, ${username}님!`);
  } else {
    console.log("이름이 입력되지 않았습니다.");
  }
  ```

### `switch`

하나의 표현식을 여러 값과 비교하여 일치하는 `case`의 코드 블록을 실행합니다. `if...else if`가 길어질 때 코드를 더 간결하게 만들 수 있습니다.

```javascript
const fruit = "사과";

switch (fruit) {
  case "사과":
    console.log("사과는 1000원입니다.");
    break; // break를 만나면 switch 문을 빠져나갑니다.
  case "바나나":
    console.log("바나나는 1500원입니다.");
    break;
  case "체리":
  case "딸기":
    console.log("체리와 딸기는 2000원입니다.");
    break;
  default: // 일치하는 case가 없을 때 실행됩니다.
    console.log("판매하지 않는 과일입니다.");
}
```

- **`break`의 중요성**: 각 `case`의 끝에 `break`를 사용하지 않으면, 일치하는 `case` 이후의 모든 `case` 코드가 `break`를 만날 때까지 순차적으로 실행됩니다(이를 "fall-through"라고 합니다).

## 2. 반복문 (Loop Statements)

### `while`

조건식이 `true`인 동안 코드 블록을 계속해서 반복 실행합니다.

```javascript
let count = 0;
while (count < 3) {
  console.log(`while 반복: ${count}`);
  count++;
}
// 출력:
// while 반복: 0
// while 반복: 1
// while 반복: 2
```

### `for`

`while`보다 더 구조화된 반복문으로, 초기화, 조건식, 증감식을 한 줄에 작성하여 반복을 제어합니다.

```javascript
for (let i = 0; i < 3; i++) {
  console.log(`for 반복: ${i}`);
}
// 출력:
// for 반복: 0
// for 반복: 1
// for 반복: 2
```

### `for...of`

#for_of

ES6에서 도입된 반복문으로, 배열, 문자열, Map, Set 등 **반복 가능한(iterable)** 객체의 **값(value)**을 순서대로 순회합니다.

```javascript
const fruits = ["사과", "바나나", "체리"];

for (const fruit of fruits) {
  console.log(fruit);
}
// 출력:
// 사과
// 바나나
// 체리
```

### `for...in`

#for_in

객체(Object)의 **속성 키(key)**를 순회하기 위해 사용됩니다.

```javascript
const user = {
  name: "홍길동",
  age: 30,
  city: "서울",
};

for (const key in user) {
  console.log(`${key}: ${user[key]}`);
}
// 출력:
// name: 홍길동
// age: 30
// city: 서울
```

- **주의**: `for...in`은 배열에도 사용할 수 있지만, 배열의 인덱스(문자열로 반환됨)와 예상치 못한 프로토타입 체인의 속성까지 순회할 수 있어 권장되지 않습니다. **배열 순회에는 `for...of` 또는 배열 메서드(`forEach`, `map` 등)를 사용하는 것이 좋습니다.**
