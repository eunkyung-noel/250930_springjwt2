# 03. 폼(Form)과 입력(Input)

웹의 핵심 기능 중 하나는 사용자와의 상호작용입니다. `폼(Form)`은 로그인, 회원가입, 검색, 게시글 작성 등 사용자가 서버로 데이터를 전송할 수 있게 하는 창구 역할을 합니다.

#학습목표

- `<form>` 태그의 `action`과 `method` 속성을 이해하고 데이터를 전송하는 기본 폼을 만들 수 있습니다.
- `<input>` 태그의 다양한 `type`을 사용하여 텍스트, 비밀번호, 체크박스, 라디오 버튼 등을 구현할 수 있습니다.
- `<label>` 태그를 사용하여 웹 접근성을 준수하는 입력 필드를 만들 수 있습니다.
- `<textarea>`, `<select>`, `<button>` 등 다른 폼 관련 요소들을 활용할 수 있습니다.

---

## 1. `<form>` 태그: 데이터 전송의 시작

#폼 #form #액션 #action #메서드 #method

`<form>` 태그는 모든 입력 요소들을 감싸는 컨테이너입니다. 사용자가 입력한 데이터를 어디로, 어떻게 보낼지 정의합니다.

- **`action` 속성**: 폼 데이터가 전송될 서버의 URL을 지정합니다.
- **`method` 속성**: 데이터를 전송하는 HTTP 메서드를 지정합니다.
  - **`get`**: 데이터를 URL의 쿼리스트링(e.g., `?name=John&age=30`)에 붙여서 전송합니다. 간단한 검색 등 민감하지 않은 데이터를 보낼 때 사용하며, URL에 데이터가 노출됩니다.
  - **`post`**: 데이터를 HTTP 요청 본문(Body)에 담아 전송합니다. 로그인, 회원가입 등 민감한 정보나 용량이 큰 데이터를 보낼 때 사용하며, 데이터가 URL에 노출되지 않습니다.

### GET 방식 예제

```html
<!-- "JAVA"를 검색하면 https://search.naver.com/search.naver?query=JAVA 로 이동 -->
<form action="https://search.naver.com/search.naver" method="get">
  <input type="text" name="query" />
  <button>네이버 검색</button>
</form>
```

## 2. `<input>` 태그와 다양한 `type`

#인풋 #input #타입 #type

`<input>` 태그는 사용자가 데이터를 입력할 수 있는 다양한 형태의 필드를 만듭니다. `type` 속성에 따라 모양과 기능이 크게 달라집니다.

### 주요 `type` 속성 값

| `type`     | 설명                                      | 주요 속성                          |
| ---------- | ----------------------------------------- | ---------------------------------- |
| `text`     | 한 줄 텍스트 입력 필드                    | `name`, `placeholder`, `maxlength` |
| `password` | 비밀번호 입력 필드 (입력값이 가려짐)      | `name`, `placeholder`              |
| `email`    | 이메일 형식 입력 필드 (기본 검증)         | `name`, `placeholder`              |
| `number`   | 숫자 입력 필드                            | `name`, `min`, `max`, `step`       |
| `checkbox` | 다중 선택 가능한 체크박스                 | `name`, `value`, `checked`         |
| `radio`    | 여러 개 중 하나만 선택 가능한 라디오 버튼 | `name`, `value`, `checked`         |
| `file`     | 파일 첨부 필드                            | `name`, `accept`                   |
| `date`     | 날짜 선택 필드 (달력 UI)                  | `name`                             |
| `hidden`   | 사용자에게 보이지 않는 숨겨진 데이터      | `name`, `value`                    |
| `submit`   | 폼을 전송하는 버튼                        | `value` (버튼 텍스트)              |
| `reset`    | 폼의 모든 입력 값을 초기화하는 버튼       | `value` (버튼 텍스트)              |
| `button`   | 일반 버튼 (주로 JavaScript와 함께 사용)   | `value` (버튼 텍스트)              |

- **`name` 속성**: 서버로 데이터를 전송할 때, 각 입력 값의 '키(key)' 역할을 합니다. 서버는 이 `name`을 기준으로 데이터를 식별하므로 **매우 중요**합니다.
- **`placeholder` 속성**: 입력 필드에 사용자가 어떤 값을 입력해야 하는지 알려주는 힌트 텍스트입니다.
- **`value` 속성**: 입력 필드의 초기값 또는 `checkbox`, `radio`에서 선택 시 서버로 전송될 값을 지정합니다.

## 3. `<label>` 태그와 웹 접근성

#라벨 #label #웹접근성 #accessibility

`<label>` 태그는 `input` 요소의 제목 역할을 합니다. 사용자가 라벨을 클릭했을 때 해당 `input` 요소에 포커스가 가도록 하여 사용자 경험을 향상시키고, 스크린 리더가 어떤 입력 필드인지 명확히 알려주어 웹 접근성을 높입니다.

### 연결 방법

1.  **`for` 속성 사용 (명시적 연결)**: `<label>`의 `for` 속성 값과 `<input>`의 `id` 속성 값을 일치시킵니다. **가장 권장되는 방법**입니다.
2.  **`input` 감싸기 (암시적 연결)**: `<label>` 태그 안에 `<input>` 태그를 넣습니다.

### 실습 예제: `07_form_basic.html`

```html
<!DOCTYPE html>
<html lang="ko">
  <head>
    <meta charset="UTF-8" />
    <title>기본 폼</title>
  </head>
  <body>
    <h1>회원가입</h1>
    <form action="/register" method="post">
      <!-- 1. for와 id를 이용한 명시적 연결 -->
      <div>
        <label for="user-id">아이디:</label>
        <input
          type="text"
          id="user-id"
          name="userId"
          placeholder="아이디를 입력하세요"
        />
      </div>

      <!-- 2. input을 감싸는 암시적 연결 -->
      <div>
        <label
          >비밀번호:
          <input
            type="password"
            name="userPw"
            placeholder="비밀번호를 입력하세요"
          />
        </label>
      </div>

      <button type="submit">가입하기</button>
    </form>
  </body>
</html>
```

## 4. 그 외 다양한 폼 요소

### `<textarea>`

여러 줄의 텍스트를 입력받을 때 사용합니다. (예: 게시글 본문, 긴 주소)

```html
<label for="content">내용:</label>
<textarea id="content" name="content" rows="5" cols="30"></textarea>
```

### `<select>`와 `<option>`

미리 정의된 여러 옵션 중 하나 또는 여러 개를 선택할 수 있는 드롭다운 목록이나 리스트 박스를 만듭니다.

```html
<label for="job">직업:</label>
<select id="job" name="job">
  <option value="">-- 선택하세요 --</option>
  <option value="developer">개발자</option>
  <option value="designer">디자이너</option>
  <option value="planner">기획자</option>
</select>
```

### `<button>`

클릭 가능한 버튼을 만듭니다. `type` 속성에 따라 기능이 달라집니다.

- `type="submit"`: 폼 데이터를 `action`에 지정된 주소로 전송합니다. (기본값)
- `type="reset"`: 폼 내의 모든 입력 값을 초기 상태로 되돌립니다.
- `type="button"`: 아무런 기본 동작이 없는 일반 버튼. JavaScript와 함께 사용하여 특정 기능을 실행할 때 주로 사용합니다.

```html
<button type="submit">전송</button>
<button type="reset">초기화</button>
<button type="button" onclick="alert('Hello!')">클릭</button>
```

### 실습 예제: `08_form_advanced.html`

```html
<!DOCTYPE html>
<html lang="ko">
  <head>
    <meta charset="UTF-8" />
    <title>다양한 폼 요소</title>
  </head>
  <body>
    <form action="/survey" method="post">
      <fieldset>
        <legend>설문조사</legend>

        <p>
          관심 분야 (다중 선택 가능):<br />
          <label
            ><input type="checkbox" name="interest" value="sports" />
            운동</label
          >
          <label
            ><input type="checkbox" name="interest" value="music" /> 음악</label
          >
          <label
            ><input type="checkbox" name="interest" value="travel" />
            여행</label
          >
        </p>

        <p>
          연령대 (하나만 선택):<br />
          <label><input type="radio" name="age" value="10s" /> 10대</label>
          <label><input type="radio" name="age" value="20s" /> 20대</label>
          <label><input type="radio" name="age" value="30s" /> 30대 이상</label>
        </p>

        <p>
          <label for="job">직업:</label>
          <select id="job" name="job">
            <option value="student">학생</option>
            <option value="employee">직장인</option>
            <option value="etc">기타</option>
          </select>
        </p>

        <p>
          <label for="feedback">의견:</label><br />
          <textarea
            id="feedback"
            name="feedback"
            rows="4"
            placeholder="자유롭게 의견을 남겨주세요."
          ></textarea>
        </p>

        <button type="submit">제출하기</button>
      </fieldset>
    </form>
  </body>
</html>
```

- `<fieldset>`: 관련된 폼 요소들을 그룹화합니다.
- `<legend>`: `<fieldset>` 그룹의 제목을 정의합니다.
