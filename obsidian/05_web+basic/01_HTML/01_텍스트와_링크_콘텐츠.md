# 01. 텍스트와 링크 콘텐츠

HTML에서 가장 기본이 되는 콘텐츠는 텍스트입니다. 이 문서에서는 웹 페이지의 내용을 구성하는 다양한 텍스트 관련 태그와, 페이지 간을 연결하는 하이퍼링크 태그에 대해 알아봅니다.

#학습목표

- 제목(`<h1>`~`<h6>`)과 단락(`<p>`) 태그를 사용하여 문서의 기본 구조를 잡을 수 있습니다.
- `<strong>`, `<em>`, `<code>` 등 인라인 텍스트 태그로 콘텐츠를 의미에 맞게 강조할 수 있습니다.
- 순서가 있는 목록(`<ol>`)과 순서가 없는 목록(`<ul>`)을 만들 수 있습니다.
- `<a>` 태그를 사용하여 다른 페이지로 이동하는 하이퍼링크를 생성할 수 있습니다.

---

## 1. 제목(Headings)과 단락(Paragraph)

#제목 #heading #단락 #paragraph

- **`<h1>` ~ `<h6>`**: 제목을 나타내는 태그입니다. `<h1>`이 가장 중요한 최상위 제목이며, `<h6>`으로 갈수록 중요도가 낮아집니다. 검색 엔진은 이 태그들을 사용하여 문서의 구조와 주제를 파악하므로, 의미에 맞게 사용하는 것이 중요합니다.

- **`<p>`**: 단락(Paragraph)을 만드는 태그입니다. 하나의 문단은 하나의 `<p>` 요소로 감싸는 것이 일반적입니다. 브라우저는 `<p>` 태그 위아래에 기본적으로 약간의 여백을 추가하여 단락을 구분합니다.

- **`<br>`**: 줄 바꿈(Line Break) 태그입니다. 단락을 나누지 않고 강제로 줄만 바꾸고 싶을 때 사용합니다. 종료 태그가 없는 빈 태그(Empty Tag)입니다.

### 실습 예제: `01_headings_paragraphs.html`

```html
<!DOCTYPE html>
<html lang="ko">
  <head>
    <meta charset="UTF-8" />
    <title>제목과 단락</title>
  </head>
  <body>
    <!-- h1: 페이지의 주제, 가장 큰 제목. 보통 한 페이지에 한 번만 사용 -->
    <h1>블로그 포스트 제목</h1>

    <!-- h2: 주요 섹션 제목 -->
    <h2>1. 서론</h2>
    <p>
      이것은 첫 번째 단락입니다. 웹 페이지의 내용을 구성하는 기본 단위입니다.
    </p>
    <p>
      이것은 두 번째 단락입니다.<br />이렇게 문단 중간에 줄바꿈을 할 수도
      있습니다.
    </p>

    <h2>2. 본론</h2>
    <p>여기는 본론 내용이 들어갑니다.</p>
  </body>
</html>
```

## 2. 인라인(Inline) 텍스트 꾸미기

단락 내에서 특정 단어나 구절에 특별한 의미를 부여할 때 사용하는 태그들입니다.

#인라인 #inline #강조 #emphasis

| 태그       | 의미                                  | 시각적 표현 (기본) | 예시                                     |
| ---------- | ------------------------------------- | ------------------ | ---------------------------------------- |
| `<strong>` | **중요한** 콘텐츠 (Strong Importance) | 굵게               | `<strong>경고:</strong> 위험합니다.`     |
| `<em>`     | *강조*하는 텍스트 (Emphasis)          | 기울임꼴           | `이 부분을 *특히* 잘 읽어주세요.`        |
| `<b>`      | **굵은** 글씨 (Bold)                  | 굵게               | `제품명: <b>AwesomeWidget</b>`           |
| `<i>`      | _기울임_ 글씨 (Italic)                | 기울임꼴           | `학명: <i>Homo sapiens</i>`              |
| `<mark>`   | 하이라이트, <mark>형광펜</mark> 표시  | 노란 배경          | `검색 결과에서 <mark>키워드</mark> 표시` |
| `<code>`   | 컴퓨터 `코드` 조각                    | 고정폭 글꼴        | `<code>System.out.println()</code>`      |
| `<small>`  | <small>작은</small> 글씨 (부가 정보)  | 작게               | `저작권: <small>&copy; 2024</small>`     |

**`<strong>` vs `<b>`**: `<strong>`은 의미론적으로 '중요함'을 나타내지만, `<b>`는 특별한 의미 없이 단순히 글자를 굵게 표현할 때 사용합니다. 스크린 리더는 `<strong>` 태그를 더 강조해서 읽어줄 수 있습니다.

## 3. 목록(Lists)

콘텐츠 항목들을 나열할 때 사용하며, 순서 유무에 따라 두 가지로 나뉩니다.

#목록 #list #유엘 #오엘

- **`<ul>` (Unordered List)**: 순서가 중요하지 않은 목록을 만듭니다. 각 항목은 `<li>`(List Item) 태그로 감쌉니다.
- **`<ol>` (Ordered List)**: 순서가 중요한 목록을 만듭니다. 마찬가지로 각 항목은 `<li>` 태그를 사용하며, 브라우저가 자동으로 번호를 매겨줍니다.

### 실습 예제: `02_lists.html`

```html
<!DOCTYPE html>
<html lang="ko">
  <head>
    <meta charset="UTF-8" />
    <title>목록 만들기</title>
  </head>
  <body>
    <h1>장보기 목록 (순서 없음)</h1>
    <ul>
      <li>우유</li>
      <li>계란</li>
      <li>빵</li>
    </ul>

    <h1>라면 끓이는 법 (순서 있음)</h1>
    <ol>
      <li>물을 550ml 끓인다.</li>
      <li>면과 스프를 넣는다.</li>
      <li>4분 30초 더 끓인다.</li>
      <li>맛있게 먹는다.</li>
    </ol>
  </body>
</html>
```

## 4. 하이퍼링크(Hyperlinks)

#링크 #link #하이퍼링크 #hyperlink

`<a>` 태그(Anchor의 약자)는 다른 웹 페이지, 같은 페이지 내의 특정 위치, 파일 다운로드 등 다양한 리소스로 연결되는 링크를 만듭니다.

- **`href` (Hypertext Reference) 속성**: 링크의 목적지 URL을 지정하는 **필수 속성**입니다.
- **`target` 속성**: 링크를 여는 방법을 지정합니다.
  - `_self`: 현재 창에서 열기 (기본값)
  - `_blank`: 새 탭이나 새 창에서 열기
- **`rel="noopener noreferrer"`**: `target="_blank"` 사용 시 보안 취약점을 방지하기 위해 함께 사용하는 것이 권장됩니다.

### 실습 예제: `03_links.html`

```html
<!DOCTYPE html>
<html lang="ko">
  <head>
    <meta charset="UTF-8" />
    <title>하이퍼링크</title>
  </head>
  <body>
    <h1>다양한 링크</h1>

    <!-- 다른 웹사이트로 이동 -->
    <p><a href="https://www.google.com">구글로 이동하기</a></p>

    <!-- 새 탭에서 열기 -->
    <p>
      <a
        href="https://developer.mozilla.org/ko/"
        target="_blank"
        rel="noopener noreferrer"
        >MDN 웹 문서 (새 탭)</a
      >
    </p>

    <!-- 같은 폴더 내의 다른 HTML 파일로 이동 -->
    <p><a href="01_headings_paragraphs.html">제목과 단락 예제 보기</a></p>

    <!-- 이메일 링크 -->
    <p><a href="mailto:contact@example.com">문의 메일 보내기</a></p>

    <!-- 전화 걸기 링크 (모바일) -->
    <p><a href="tel:010-1234-5678">전화 걸기</a></p>
  </body>
</html>
```

다음 문서에서는 이미지, 비디오 등 미디어 콘텐츠를 삽입하는 방법과 페이지의 전체적인 구조를 잡는 레이아웃 태그에 대해 알아보겠습니다.
