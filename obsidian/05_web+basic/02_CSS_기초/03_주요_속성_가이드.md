# 03. 주요 속성 가이드: 박스 모델, 글꼴, 색상

CSS에는 수많은 속성이 있지만, 웹 페이지의 기본 모양과 느낌을 결정하는 몇 가지 핵심 속성 그룹이 있습니다. 이 문서에서는 모든 웹 요소의 레이아웃 기초가 되는 **박스 모델(Box Model)**과 타이포그래피의 핵심인 **글꼴(Font)**, 그리고 시각적 표현의 기본인 **색상(Color)** 관련 속성들을 알아봅니다.

#학습목표

- CSS 박스 모델의 네 가지 구성 요소(content, padding, border, margin)를 이해하고 제어할 수 있습니다.
- `font-family`, `font-size`, `font-weight` 등 주요 글꼴 관련 속성을 사용하여 텍스트 스타일을 지정할 수 있습니다.
- `color`와 `background-color` 속성을 사용하여 요소의 전경색과 배경색을 변경할 수 있습니다.

---

## 1. 박스 모델 (The Box Model)

#박스모델 #box_model #레이아웃 #layout

CSS에서 모든 HTML 요소는 사각형의 **박스(Box)**로 간주됩니다. 이 박스는 네 가지 영역으로 구성되며, 이를 **박스 모델**이라고 합니다. 박스 모델을 이해하는 것은 CSS 레이아웃의 첫걸음입니다.

```mermaid
graph TD
    subgraph Margin (바깥 여백)
        subgraph Border (테두리)
            subgraph Padding (안쪽 여백)
                Content["Content<br>콘텐츠 영역<br>(width, height)"]
            end
        end
    end
    style Margin fill:#f9f9f9,stroke:#ccc,stroke-dasharray: 5 5
    style Border fill:#e6e6e6,stroke:#999
    style Padding fill:#d9d9d9,stroke:#666
    style Content fill:#fff,stroke:#333
```

- **Content (콘텐츠 영역)**: 텍스트, 이미지 등 요소의 실제 내용이 표시되는 영역입니다. `width`와 `height` 속성으로 크기를 제어합니다.
- **Padding (안쪽 여백)**: 콘텐츠 영역과 테두리(Border) 사이의 공간입니다. 요소 내부에 여유 공간을 만들 때 사용합니다.
- **Border (테두리)**: 패딩의 바깥쪽을 감싸는 선입니다. 두께, 스타일, 색상을 지정할 수 있습니다.
- **Margin (바깥 여백)**: 테두리의 바깥쪽 영역으로, 다른 요소와의 간격을 조정할 때 사용합니다.

### 주요 박스 모델 속성

- **`width`, `height`**: 콘텐츠 영역의 너비와 높이를 지정합니다.
- **`padding`**: `padding: 10px;` (상하좌우 모두), `padding: 10px 20px;` (상하, 좌우), `padding-top: 5px;` (특정 방향)
- **`border`**: `border: 1px solid black;` (두께, 스타일, 색상)
- **`margin`**: `margin: 10px;` (상하좌우 모두), `margin: 0 auto;` (블록 요소를 수평 중앙 정렬)
- **`box-sizing`**:
  - `content-box` (기본값): `width`와 `height`가 콘텐츠 영역만 포함합니다. (전체 너비 = `width` + `padding`*2 + `border`*2)
  - `border-box`: `width`와 `height`가 패딩과 테두리를 포함합니다. 레이아웃을 예측하기 훨씬 쉬워져서 **대부분의 프로젝트에서 모든 요소에 `border-box`를 적용하는 것이 일반적**입니다.

```css
/* 모든 요소에 border-box를 적용하는 일반적인 방법 */
* {
  box-sizing: border-box;
}
```

## 2. 글꼴 (Font) 속성

#글꼴 #font #타이포그래피 #typography

웹 페이지의 가독성과 디자인 인상을 결정하는 중요한 요소입니다.

- **`font-family`**: 글꼴을 지정합니다. 사용자의 컴퓨터에 해당 글꼴이 없을 경우를 대비해 여러 글꼴을 쉼표로 구분하여 나열합니다 (Fallback).
  ```css
  /* 첫 번째 글꼴이 없으면 다음 글꼴을, 그것도 없으면 기본 sans-serif 글꼴을 사용 */
  font-family: "Helvetica Neue", Arial, sans-serif;
  ```
- **`font-size`**: 글자 크기를 지정합니다. `px`, `em`, `rem`, `%` 등의 단위를 사용합니다.
  - `16px`: 절대적인 픽셀 단위.
  - `1.2rem`: 루트 요소(`html`)의 글자 크기를 기준으로 한 상대적인 배율. (가장 권장)
  - `1.2em`: 부모 요소의 글자 크기를 기준으로 한 상대적인 배율.
- **`font-weight`**: 글자 두께를 지정합니다. `normal`(400), `bold`(700) 또는 100~900 사이의 숫자 값을 사용합니다.
- **`font-style`**: 글자 스타일을 지정합니다. `normal`, `italic`, `oblique` 등이 있습니다.
- **`line-height`**: 줄 간격을 지정합니다. 가독성을 위해 보통 글자 크기의 1.5~1.7배 정도를 사용합니다. (`line-height: 1.6;`)
- **`letter-spacing`**: 글자 사이의 간격(자간)을 조절합니다.
- **`text-align`**: 텍스트의 수평 정렬을 지정합니다. `left`, `right`, `center`, `justify` (양쪽 정렬).

## 3. 색상 (Color) 속성

#색상 #color #배경색 #background_color

- **`color`**: 글자 등 전경(Foreground)의 색상을 지정합니다.
- **`background-color`**: 요소의 배경색을 지정합니다.

### 색상 값 표현 방법

| 방법         | 예시                         | 설명                                          |
| ------------ | ---------------------------- | --------------------------------------------- |
| **키워드**   | `red`, `blue`, `transparent` | 미리 정의된 색상 이름.                        |
| **HEX**      | `#FF0000`, `#F00`            | 16진수 색상 코드. 가장 널리 사용됩니다.       |
| **RGB/RGBA** | `rgb(255, 0, 0)`             | Red, Green, Blue 값 (0~255).                  |
|              | `rgba(255, 0, 0, 0.5)`       | RGB에 Alpha(투명도, 0~1)를 추가.              |
| **HSL/HSLA** | `hsl(0, 100%, 50%)`          | Hue(색상), Saturation(채도), Lightness(명도). |

### 실습 예제: `properties_practice.html`

```html
<!DOCTYPE html>
<html lang="ko">
  <head>
    <title>주요 속성 실습</title>
    <style>
      .card {
        /* 박스 모델 */
        width: 300px;
        padding: 20px;
        border: 1px solid #ccc;
        border-radius: 8px; /* 모서리를 둥글게 */
        margin: 20px;
        box-shadow: 0 4px 8px rgba(0, 0, 0, 0.1); /* 그림자 효과 */

        /* 색상 */
        background-color: #f9f9f9;

        /* 글꼴 */
        font-family: "Verdana", sans-serif;
      }
      .card h2 {
        color: #333;
        border-bottom: 2px solid #eee;
        padding-bottom: 10px;
        text-align: center;
      }
      .card p {
        color: #666;
        font-size: 16px;
        line-height: 1.6;
      }
    </style>
  </head>
  <body>
    <div class="card">
      <h2>CSS 카드 예제</h2>
      <p>이것은 박스 모델, 글꼴, 색상 속성을 사용하여 스타일링된 카드입니다.</p>
    </div>
  </body>
</html>
```

다음 문서에서는 `@font-face` 규칙을 사용하여 사용자의 컴퓨터에 설치되지 않은 글꼴, 즉 **웹폰트**를 웹 페이지에 적용하는 방법을 실습해 보겠습니다.
